Installation Instructions

1. Open the [app.py](http://app.py/) Python file in your IDE

2. Link the Python file to the working directory. In the terminal, write `cd "paste path"` to the folder that was unzipped.
Example: cd /Users/Desktop/my_app

3. Install Required Libraries. In the folder, there is a requirements.txt file.
• In the terminal, write `pip install -r requirements.txt`
• This will install all the packages needed

4. Running the app: We have built the experience on the Streamlit framework. To launch the app, write in the terminal `streamlit run [app.py](http://app.py/)`

5. This will launch the experience in your browser
